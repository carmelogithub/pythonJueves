API (2)
1. conjunto de herramienta

java21 tiene librerías (archivos con clases métodos,..funcionalides)

2. API Rest
tienes datos en una fuente de datos(base de datos, archivo, repositorio, data Lake...)
quieres acceder a esos datos(2)
a. librería de conexión, usuario y contraseña, base de datos, tabla
b. te doy los datos con un sistema APIRest (json)
